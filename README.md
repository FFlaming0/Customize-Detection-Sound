# 自定义敌人发现玩家时的音效 - 逃离鸭科夫MOD

## 简介

当敌方发现玩家时会额外播放Ciallo～(∠・ω< )⌒★

我保留了鸭子原始的嘎嘎声，这样你才知道玩的是鸭科夫。

## 最新更新
1. 添加按F8播放Ciallo
2. 支持自定义各种类型的敌方单位的音效，可以差分化(mod默认音频全是一样的，如需要可以自行修改)
3. 支持同一类型敌方单位随机播放多种音效(自行配置)
具体内容请查看最新”改动说明“

因精力有限且本人游玩进度较慢，可能会有部分敌方单位未检测到，欢迎大家开启控制台mod(创意工坊搜索)来检查是否有其他类型的敌方单位，并在指定讨论帖中反馈。反馈格式见讨论

## 制作过程 && 个性化方式：
dnSpy反编译寻找AICharacterController类以了解AI控制逻辑，对AI发现玩家后新增逻辑以播放自定义音频

在MOD路径中有一个audios文件夹，这其中可以自行替换为其他的语音，但是命名格式固定，具体见改动说明
路径(如果您不知道这是什么意思，请忽略)：
$(Steam安装路径)\steamapps\workshop\content\3167020\3592615697\audios

支持通过ModConfig的图形化界面和直接修改配置文件两种方式进行参数修改
如未订阅ModConfig，本mod功能不受影响，但仅支持通过配置文件进行修改
注意在游戏内MOD页面中将ModConfig放在本MOD上面，否则可能会加载失败
配置文件路径：(如果您不知道这是什么意思，请忽略)
$(Steam安装路径)\steamapps\workshop\content\3167020\3592615697\CialloDetect.cfg

## 配置说明(以下具体参数为默认设置)：
volume=1.0 -> 音量调节，0.0(静音) ~ 1.0(最大)
clock=3.0 -> 触发间隔(单位s)，在间隔内触发音效至上限时进入cd
counter=2 -> 触发上限次数
cd=5.0 -> 触发cd(单位s)
